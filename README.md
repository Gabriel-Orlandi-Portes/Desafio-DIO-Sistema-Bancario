# Sistema Bancário em Python

Projeto desenvolvido como parte do desafio de projeto "Criando um Sistema Bancário com Python" da DIO (Digital Innovation One).

## Descrição

Este projeto consiste em um sistema bancário simples desenvolvido em Python utilizando conceitos fundamentais da linguagem, como estruturas condicionais, laços de repetição e manipulação de variáveis.

## Funcionalidades

* Realizar depósitos
* Realizar saques
* Limite de R$ 500,00 por saque
* Limite de 3 saques diários
* Consultar extrato
* Consultar saldo atual

## Tecnologias Utilizadas

* Python 3

## Como Executar

1. Clone o repositório:

   ```bash
   git clone https://github.com/Gabriel-Orlandi-Portes/Desafio-DIO-Sistema-Bancario
   ```

2. Execute o programa:

   ```bash
   python sistema_bancario.py
   ```

## Sobre o Desafio

Este projeto foi desenvolvido como parte do desafio de projeto "Criando um Sistema Bancário com Python" da plataforma DIO, com o objetivo de praticar conceitos básicos de programação e lógica utilizando Python.
